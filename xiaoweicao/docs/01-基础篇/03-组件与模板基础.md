### 组件与模板基础

常用基础组件：`view`、`text`、`image`、`button`、`input`、`picker`。

列表渲染与事件：

```vue
<template>
  <view class="list">
    <view v-for="(item, index) in items" :key="index" @click="onTap(item)">
      {{ item.title }}
    </view>
  </view>
</template>

<script setup>
import { ref } from 'vue'
const items = ref([{ title: 'A' }, { title: 'B' }])
function onTap(item) { uni.showToast({ title: item.title }) }
</script>

<style scoped>
.list { padding: 24rpx }
</style>
```
