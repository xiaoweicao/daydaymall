- 理解多端适配和性能优化技巧

## 🚀 快速开始

### 1. 环境准备

```bash
# 安装 Node.js (推荐 16+)
# 安装 HBuilderX 或使用 CLI

# 全局安装 Vue CLI
npm install -g @vue/cli

# 安装 uni-app 官方 CLI
npm install -g @dcloudio/uvm
uvm
```

### 2. 创建项目

```bash
# 使用 HBuilderX 创建
# 或使用命令行
npx degit dcloudio/uni-preset-vue#vite-ts my-uniapp-project
cd my-uniapp-project
npm install
```

### 3. 运行项目

```bash
# H5
npm run dev:h5

# 微信小程序
npm run dev:mp-weixin

# App
npm run dev:app
```
